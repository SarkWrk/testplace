*.Round(input, places?) : number

Takes a number and rounds it to the nearest integer. An optional places overload can be specified to round to a specific amount of decimal places. Rounding to a negative amount of decimal places may result in unexpected behaviour and is recommended not to be done.

Note:

  • This function relies on *.NumberManipulation.oom:() to function.

Inputs:

Value Type Explanation
input number The number that should be rounded.
places number? How many decimal places to round input to.

Returns:

Value Type Explanation
number number input rounded to the nearest integer unless places is specified.

*.Truncate(input, places) : (leading, magnitude)

Truncates a number to places amount of leading digits. Returns a tuple of the leading digits of the truncated number and its order of magnitude.

Note:

  • leading is returned as the leading digits x10^-1.
  • This function relies on *.NumberManipulation.oom:() to function.

Inputs:

Value Type Explanation
input number The number that should be truncated.
places number How many leading digits input should be truncated to.

Returns:

Value Type Explanation
leading number The leading digits of the truncated number. Note: This value is x10^-1
magnitude number The order of magnitude of the truncated number.

*.Vector3ToCFrame(position, rotation?) : CFrame

Creates a new CFrame using position as the position of the CFrame. An optional rotation overload can be specified to add a rotation to the CFrame.

Inputs:

Value Type Explanation
position Vector3 The positional component that the resulting CFrame should have.
rotation Vector3? An optional overload to specify the rotation the CFrame should have.

Returns:

Value Type Explanation
CFrame CFrame A CFrame with position position. If rotation is specified, it will have that rotation applied to it.

*.Vector3ToString(vector, separator, round?, places?) : string

Converts a Vector3 to a string, separating each component with separator. An optional round overload can be added to specify whether each component should be rounded. If round is specified, places can be added to specify to how many places each component should be rounded to.

Note:

  • This function relies on *.NumberManipulation.Round() to function.

Inputs:

Value Type Explanation
vector Vector3 The Vector3 that should be turned into a string.
separator string A string that separates each component of the Vector3.
round boolean? Whether each component of vector should be rounded.
places number? How many places to round each component to. Note: only takes effect if round is true.

Returns:

Value Type Explanation
string string A string formatted as: vector.Xseparatorvector.Yseparatorvector.Z.

*.CFrameAnglesToVector3(converted, round?, places?) : Vector3

Converts a CFrame's rotation into a Vector3 where each of the components relate to the rotational component of the CFrame. If round is added, the rotational components will be rounded. If round is added, then places can be added to specify how many places to round to.

Note:

  • This function relies on *.NumberManipulation.Round() to function.

Inputs:

Value Type Explanation
converted CFrame The CFrame who's rotation should be converted into a Vector3.
round boolean? Whether each component of the rotation of converted should be rounded.
places number? How many places to round converted to. Note: only functions if round is true.

Returns:

Value Type Explanation
Vector3 Vector3 A Vector3 with components of the rotational components of converted.
